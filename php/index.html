Stock Analyzer Application

This is a PHP web scraper that scrapes data from Yahoo finance site into a database which is then fed to the UI as JSON. The JSON is then converted to a graph using Google Charts.

Technologies Used:

  • PHP 5
  • MySQL Server
  • Google Charts
  • Apache2 Web Server
  • XAMPP for local development
  • Ubuntu AWS EC2 Instance in the cloud

How does it work:

  • A script called stockDownloader.php reads the ticker symbols contained in a text file
  • With each symbol, the scraper gets the data of the stock price from Yahoo finance using a date range.
  • This data is then saved into a folder on the server as a text file per ticker symbol
  • The stockDownloader script then takes the data from each ticker symbol file and strips out the first row headers first and then saves the data to a table in the database using the ticker symbol
  • On the user interface, each page runs a query that fetches the data from the database based on the ticker symbol
  • Each row is fetched as an associative array in PHP and then converted to JSON
  • The JSON data is then fed into Google charts on the UI using the closing price as the vertical axis and the date as the horizontal axis.
