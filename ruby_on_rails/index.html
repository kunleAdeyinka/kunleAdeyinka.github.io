Bookstore Application

This is a simple web bookstore built using Ruby on Rails. The site allows users to browse books by category. It allows a user to enter the author and publisher information and view books based on the author and publisher. The layout of the site is built using Bootstrap. Books, Categories, Authors and Publishers can be added from the front end. Didn't include a login and user registration section. Will add those later.

Technologies Used:

  • Ruby On Rails
  • MySQL Server
  • Bootstrap
  • Apache2 Web Server
  • XAMPP for local development

  • Demo URL on AWS
  • Github URL
