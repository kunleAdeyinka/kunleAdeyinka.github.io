Project 5: Marvel Universe Search

Iteration 1: Requirements

Overview

Vision Statement

The goal is to create a web based search engine that allows users to query it to get information about the vast universe of Marvel characters, comics, movies and television shows. The engine is based mostly on Marvel Comics API and will be used to process queries and present the results to users. Due to the rate limit of calls to the API in a day, some form of caching is going to be used to try to limit the API calls and to improve performance.

Also data will be scraped from various other sources on the web to glean further insights. A list of top searches will be shown and current and future Marvel projects will be displayed for users to view for further information.
