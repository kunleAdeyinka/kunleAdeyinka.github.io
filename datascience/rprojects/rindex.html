What is R

From the R project, "R is a language and environment for statistical computing and graphics". It is an interpreted language (instructions executed directly without compilation to machine language) and is an implementation of the S statistical language from Bell Labs in the 70's. It is a domain specific language and is not intended for general purpose usage.

R is also supported by CRAN, the Comprehensive R Archive Network (http://cran.r-project.org/). CRAN contains an archive of previous versions of R and hundreds of freely downloadable software packages that extend the capabilities of R.

RStudio offers the tools and services needed to work with R in an IDE. This allows ease of use in writing R code and organizing a project easily.
