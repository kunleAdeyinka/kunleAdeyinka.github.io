Automobile Data Analysis

This is project is to provide analysis of auto mobile fuel economy data. The steps to perform this task are as follows:

  • Acquisition of data
  • Exploration and understanding the data
  • Munging, wrangling, and manipulation
  • Analysis and modeling
  • Communication and Visualization

Acquiring the data

The dataset that will be used contains fuel efficiency performance metrics, measured in miles per gallon (MPG) over time, for most makes and models of automobiles available in the U.S since 1984. The data was made available by the U.S Deparment of Energy and the U.S Environmental Protection Agency.

The data also contains several features and attribues of the automobiles listed to allow for summarization and grouping of data to determine which groups tend to have better fuel efficiency historically and how this has changed over the years. The dataset can be obtained from http://www.fueleconomy.gov/feg/epadta/vehicles.csv.zip and information about the variables in the dataset can be found at http://www.fueleconomy.gov/feg/ws/index.shtml#vehicle.

  • downlod the dataset from the http://www.fueleconomy.gov/feg/epadta/vehicles.csv.zip
  • Unzip the vehicles.csv and move it to your working directory
  • Open the vehicles.csv file with MS Excel or any text editor. Scroll through some of the data and get a feel for it
  • Navigate to the http://www.fueleconomy.gov/feg/ws/index.shtml#vehicle
  • Select and copy all the text below the vehicle heading under Data Description and paste it into a text file. Don't include the emissions headings. Save this file in your working directory and call it varlabels.txt
