Data Science

Data science involves using mathematics and algorithmic techniques to solve some complex problem, levering data to glean information about what lies beneath the surface.

It is a multidisciplinary field where skill sets are lie in three main competencies:

  • Mathematics - this involves viewing data through a quantitative or statistical lens
  • Technology/Hacking Skills - involves using technical skills to build things and find clever solutions to problems
  • Strategy Acumen - having a keen sense of how to dissect and approach problems with regards to the business or project

Here are the programming languages I’m going to focus on for my data science learning. These are also going to be used in my various projects:

  • R Language
  • Python

Before writing any code or installing any software, there are key repeatable steps that I will use for data science or analysis throughout this tutorial. The following five steps are key:

  1. Acquisition : The first step is to acquire the data from a variety of sources, including relational databases, NoSQL and document stores, web scraping, and distributed databases such as HDFs on a Hadoop platform, RESTful APIs, flat files and in some crazy cases PDF files.
  2. Exploration and Understanding : The second involves trying to understand the data to be used and how it was collected
  3. Munging, wrangling and manipulation : This means cleaning and getting the data in the format needed for analysis
  4. Analysis and modelling : Here one gets to explore the statistical relationships between the variables in the data and pulls out tricks to cluster, categorize or classify the data and create predictive models.
  5. Communication and visualization : Putting the data in a compelling format and structure to inform users. The product produced can be a one of report or a scalable web product that will be used interactively by millions.
