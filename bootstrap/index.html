Sample Photo Sharing App Website

This is a simple responsive website built using Twitter's Bootstrap. It is for a fictional photo sharing mobile application. Features used from Bootstrap include: navigation, jumbotron, accordion menu, the 12 grid system, image classes, button classes, Glyphicons, Responsive media query.

Technologies Used:

  • Bootstrap
  • Apache2 Web Server
  • XAMPP for local development
  • Ubuntu AWS EC2 Instance in the cloud

  • Demo URL on AWS
  • Github URL
