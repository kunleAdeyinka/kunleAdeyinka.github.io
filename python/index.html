A Simple Web Scraper

This is a simple scraper written in Python tht extracts the roster of inmates at the Boone County Jail in Missouri from the local government's website using this url https://report.boonecountymo.org/mrcjava/servlet/SH01_MP.I00290s . The script extracts the first names, last names, middle names, sex, age, city and state from the HTML table. The table contains text items such as the Details link on each row and the " " which are ignored. Once extracted, the data from each row is placed in a csv file.
Technologies Used:

  • Python
  • BeatufilSoup
  • Requests package
  • Github URL for Inmate Roster Scraper
